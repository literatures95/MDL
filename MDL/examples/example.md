# MDL 示例文档

这是一个用于演示 **MDL 编程语言** 功能的 Markdown 示例文档。

## 简介

MDL (Markdown Operation Language) 是一门专门为操作 Markdown 文档而设计的**领域特定语言（DSL）**。

### 核心特性

- 完美解析: 支持所有标准 Markdown 语法
- 精确修改: 增删改查一应俱全
- 深度分析: 统计、可读性评分、健康检查
- 格式转换: HTML / JSON / 纯文本无缝转换
- 智能存储: 拆分、序列化、反序列化

## 代码示例

```python
# 使用 MDL 加载并分析文档
load "document.md" as doc

# 打印所有标题
for h in doc.heading:
    print h.text
end

# 转换为 HTML
convert doc to html save as "output.html"
```
